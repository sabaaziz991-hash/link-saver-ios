Privacy-first positioning

A private bookmark workflow built around local storage.

LinkBook is consistently described as a local-first app in the report and project docs. The support text in this repo says data is stored locally and that internet access is used to fetch metadata for saved links.

What Uses the Network

Based on text in `support_page/index.html` and app readiness notes.

  • Fetching metadata (title, description, image) for links users choose to save.
  • Optional sync functionality for users who enable it.
  • Normal open-link behavior when the user launches original URLs.

The key privacy angle for SEO pages is clear: user-controlled saving and reading workflows, not ad-tech profiling.
